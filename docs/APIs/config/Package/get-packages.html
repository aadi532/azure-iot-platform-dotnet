    Get Packages

    API that fetches packages with or without parameters.

    Permissions

    - ReadAll

    Http Request

    GET /config/v1/packages

    Query Params

    Header Value
    packageType Package Type
    configType Config Type
    tags List of tags
    tagOperator Tag Operator to filter

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Body

    N/A

    Sample Request

    GET /config/v1/packages?PackageType=EdgeManifest&Tags=devicegroup.*&Tags=devicegroup.34b35ac5-837d-4fa4-9ab7-3cdf036a5145
