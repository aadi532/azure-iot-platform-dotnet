    Upload Package

    API that uploads package firmware.

    Permissions

    - CreatePackages

    Http Request

    POST /config/v1/packages/UploadFile

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.

    Other Attributes

    Header Value
    RequestSizeLimit 200000000

    Request Body

    FormData: JSON file
    Header Value
    uploadedFile {file}
