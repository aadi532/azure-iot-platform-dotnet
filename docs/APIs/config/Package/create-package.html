    Create Package

    API that create new package.

    Permissions

    - CreatePackages

    Http Request

    POST /config/v1/packages

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.

    Request Body

    FormData:
    Header Value
    packageName Name of the package(firmware file)
    packageType Type of the package
    version version number
    configType Configuration type of the package
    tags list of tags
    package package/firmware file
