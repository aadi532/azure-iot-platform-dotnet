    Get All Device Groups

    API that fetches all the Device groups.

    Permissions

    - ReadAll

    Http Request

    GET /config/v1/devicegroups

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Body
