    Get Logo

    API that fetches the logo of the solution.

    Permissions

    - ReadAll

    Http Request

    GET /config/v1/solution-settings/logo

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type image/svg+xml or image/png or image/jpeg

    Request Body

    N/A

    Response Headers

    Header Value
    Content-Type image/svg+xml or image/png or image/jpeg
    Name {Name}
    IsDefault true/false
    Access-Control-Expose-Headers Name,IsDefault
