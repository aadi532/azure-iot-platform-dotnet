    Get alarms for rule

    API that fetches the alarms details for specified rule.

    Permissions

    ReadAll

    Http Request

    GET /telemetry/v1/AlarmsByRule/{id}

    Parameters

    Parameter Type Description
    Id guid It represents RuleId

    Query parameters

    Query parameter Type Example Description
    from string NOW-PT24H The start of the time window for the query. ISO8601 format or "NOW-{ISO 8601 duration}"
    to string NOW The end of the time window for the query. ISO8601 format or "NOW-{ISO 8601 duration}"
    order string desc Whether to sort the result from the oldest (asc) or the most recent (desc)
    skip int 0 How many records to skip, used to paginate through the global list of alarms
    limit int 50 How many records to return, used to paginate through the global list of alarms
    devices string id1,id6,xyz A filter used to request alarms for specific devices

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Body

    N/A
