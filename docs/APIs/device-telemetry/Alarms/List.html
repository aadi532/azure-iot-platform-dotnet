    List Alarms

    Service: Device Telemetry

     GET https://{platform-url}/telemetry​/v1​/Alarms

    URI Parameters

    Name Required Type Description
    from No string Start time as ISO8601 or "NOW-{ISO 8601 duration}"
    to No string End time as ISO8601 or "NOW-{ISO 8601 duration}"
    order No string Set sort order: oldest ("asc"), newest ("desc")
    skip No string Alarms to skip. Used for pagination.
    limit No string Return count. Default 100.
    devices No string Filter used to request specific devices

    Responses

    Name Type
    200 OK AlarmListApiModel
