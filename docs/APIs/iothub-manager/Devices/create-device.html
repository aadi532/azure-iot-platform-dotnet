    Create Device

    Creates an IoT or Edge Device in IoTHub.

    Permissions

    CreateDevices permission is required to call this api.

    Http Request

    POST /iothub-manager/v1/devices

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
