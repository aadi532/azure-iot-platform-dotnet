    Get Deployment Impacted Modules

    Lists the Modules which are affected by the Deployment.

    Permissions

    ReadAll permission is required to call this api.

    Http Request

    POST /iothub-manager/v1/deployments/Modules/{id}?isLatest={isLatest}

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Query Params

    Name Value
    id Deployment Id
    isLatest true will retrieve the data of Modules using the query from request body
    false will retrieve the data from Modules from storage, since the deployment is not currently impacting the device group.

    Request Body

    string representing the query to fetch the devices.

    Response

    If successful, this method returns TwinPropertiesListApiModel in response body
