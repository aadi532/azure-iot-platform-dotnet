    Authorize - Authorize

    Authorizes user through configured Authorization Provider and redirects to B2c Uri.

    Http Request

     GET /connect/authorize

    URI Parameters

    Name Required Type Description
    redirect_uri Yes string callback url from B2C
    state Yes string Current State of auth
    client_id Yes string client ID GUID
    nonce Yes string nonce
    tenant Yes string tenant
    invite Yes string invitation string

    Responses

    Name Type Description
    200 OK
